# Yahoo Emoticons and Audibles in Discord

Bring back the golden era of Yahoo Messenger!

This project helps you use classic Yahoo emoticons and Yahoo audibles in Discord, with three easy options:

1. Install the Discord bot app and use Yahoo emoticons/audibles on any server and in any DM, for free (no Nitro required).

2. Join ready-made Discord servers that already have Yahoo emoticons integrated (requires Discord Nitro to use those servers emojis everywhere).

3. Invite the bot to your server to allow all server members to use Yahoo emoticons and audibles.

## What This Project Includes

- **Original Yahoo emoticon files:** [`assets/yahoo-emoticons/`](assets/yahoo-emoticons/)
- **Original Yahoo audible files:** [`assets/yahoo-audibles/`](assets/yahoo-audibles/)  
- **Full emoticon symbol list (name and symbol):** [`assets/yahoo-emoticons-list.xlsx`](assets/yahoo-emoticons-list.xlsx)

## Quick Install

Project install guide: [yahoo-emoticons.vercel.app](https://yahoo-emoticons.vercel.app/)

## Installation

### Option 1: Install the Discord Bot App (Free, No Nitro)

Invite link: [Install Yahoo Bot](https://discord.com/oauth2/authorize?client_id=1491076978990649525)

```text
https://discord.com/oauth2/authorize?client_id=1491076978990649525
```

How to add it:

1. Open the invite link.
2. Select **Add to my Apps**.
3. Authorize the bot.
4. Finish installation.

Once installed, the bot is easy to use and works on any server and any DM.

### Option 2: Use Integrated Server Emoticons (Nitro Required)

Join all 3 servers below to access all integrated Yahoo emoticons.
This split is required because Discord allows a maximum of 50 custom emojis per server, and this project uses 119 Yahoo emoticons in total.

- Server 1: [YahooEmoticons1](https://discord.gg/UqzwbwZ7Zg)
- Server 2: [YahooEmoticons2](https://discord.gg/X4p5VgEr8e)
- Server 3: [YahooEmoticons3](https://discord.gg/NDe8PcbmjT)

After joining, you can use those custom server emojis across Discord if you have Discord Nitro.

### Option 3: Invite the Bot to Your Server

Invite link: [Invite Yahoo Bot](https://discord.com/oauth2/authorize?client_id=1491076978990649525&scope=bot%20applications.commands&permissions=536989172)

```text
https://discord.com/oauth2/authorize?client_id=1491076978990649525&scope=bot%20applications.commands&permissions=536989172
```

How to add it:

1. Open the invite link.
2. Select a server to add the bot to.
3. Authorize the bot.
4. Finish installation.

## Compatibility

The Discord bot is **cross-platform**: it runs on Discord’s side, not on your device. You use it from the same Discord clients you already have-**desktop (PC)**, **tablet**, and **phone (mobile)**-including the web app and native apps on Windows, macOS, Linux, iOS, and Android. As long as you can open Discord and send messages or slash commands, the bot works the same way.

## Commands

- `!` message command (server messages only, not DMs):
  - Use `!` directly in any message text, including the middle of a sentence.
  - Format: `!<emoticon-name-or-symbol>` or `!all`
- `/y` slash command (servers + any DMs):
  - Option: `emoticon` (name, symbol, or `all`)
- `/ya` slash command (servers + any DMs):
  - Option: `audible` (required)
